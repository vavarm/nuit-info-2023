# nuit-info-2023

## Description du projet

Ce projet est un site web qui permet de sensibiliser les gens sur le réchauffement climatique. En démélant le vrai du faux via un petit quizz.

## Lien vers le site
- [lien](https://nuit-info-2023.vercel.app/)
